# Agent Skills

A collection of reusable skills for Claude Code that extend its capabilities with specialized expertise.

## Skills

| Skill | Description |
|-------|-------------|
| [architecture-designer](architecture-designer/) | System design, architecture reviews, ADRs, design patterns, scalability planning, and technology evaluation |
| [bruno-api-client](bruno-api-client/) | Generates Bruno API client collection files (`.bru`) from Express, Next.js, Fastify, or other API routes |

## Installation

Install individual skills using [skills.sh](https://skills.sh):

```bash
# Install a specific skill
npx skills add dotuan9x/agent-skills/architecture-designer
npx skills add dotuan9x/agent-skills/bruno-api-client
```
